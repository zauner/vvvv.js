VVVV.js Template Page

This is a very basic template page which runs the VVVV.js patch template_patch.v4p. The patch is included having 

<script language="VVVV" src="template_patch.v4p"></script>
in the HTML head.

Add the string #edit/template_patch.v4p to the URL to launch the patch editor. It will open in a popup window, so make sure popups are allowed for this page.
